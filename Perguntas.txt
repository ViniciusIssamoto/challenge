2 - O que são "function decorators" e para que servem?
    Adiciona funcionalidades extras a um objeto.
    Por exemplo: podemos colocar tipo um log antes e depois de cada função, sem deixar um código poluido.

3 - Quais as principais diferenças entre Python 2 e 3?
    A função "print" que precisa colocar "()"
    inteiro dividido por inteiro sempre era inteiro no python, agora ele retorna um float
    as strings são sempre em unicode
    o range() retorna um iterador

4 - Python é interpretado ou compilado?
    Interpreatado

5 - Porque usar virtualenv?
    Quando se trabalha com vários projetos, é o ideal para não misturar as bibliotecas.
    As vezes você tem um projeto que funciona na versão 1.1 da biblioteca x, daí você precisa mexer em um outro projeto
    que a versão da biblioteca é 2.0, então você cria uma virualenv para isolar as dependencias.
